# Asynchronously checking weather 

## Used
- Mongo
- Redis
- Postgres 
- JWT

## Todo
- gRPC
- Dockerfile 
- Swagger

## Start work
Fill config.yaml with your OpenWeatherAPI token.

## Register 

Send POST request: http://ADDR:PORT/register 

```
{
    "user_login":"hello",
    "user_password":"world"
}
```

## Login 

Send POST request: http://ADDR:PORT/login

```
{
    "user_login":"hello",
    "user_password":"world"
}
```

## Get weather forecast 

Send POST request http://ADDR:PORT/api/forecast

```
{
    "city":"tokyo"
}
```

```
{
    "weather": [
        {
            "description": "переменная облачность"
        }
    ],
    "main": {
        "temp": 26.01,
        "feels_lie": 0,
        "temp_min": 23.38,
        "temp_max": 26.86
    },
    "sys": {
        "country": "JP",
        "sunrise": 1660939409,
        "sunset": 1660987550
    },
    "name": "Япония"
}
```
